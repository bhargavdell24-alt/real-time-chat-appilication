# Real Time Chat Application

A full-stack real-time chat application built with the MERN stack.

## Tech Stack
- React (Vite)
- Node.js
- Express
- Socket.io
- MongoDB Atlas
- Zustand

## Features
- User authentication
- Real-time messaging
- Online users
- Profile avatars

## Local Setup

1. Clone the repo
2. Install dependencies
3. Add environment variables
4. Run backend and client
